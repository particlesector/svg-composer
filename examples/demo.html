About

This is a demo page for SVG Composer. The library is currently in development with stub implementations.


Once implemented, you'll be able to:

  • Add images, text, and shapes
  • Transform elements (move, rotate, scale)
  • Manage z-order and selection
  • Export to SVG or JSON
  • Undo/redo operations

Check the console for detailed output.
